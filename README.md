# ✨ Siwanad's Modern Portfolio

A premium, interactive portfolio website showcasing my journey, skills, and projects as a Frontend Developer. This project focuses on **high-end animations**, **responsive design**, and **optimal performance**.

🔗 **Live Demo:** [diamond-portfolio-gamma.vercel.app](https://diamond-portfolio-gamma.vercel.app/)

---

## 🚀 Overview

This portfolio is built to demonstrate my expertise in modern frontend technologies. It features a sleek, dark-themed UI with smooth transitions and interactive elements that reflect my commitment to creating engaging user experiences.

## ✨ Key Features

- **Advanced Animations:** Implemented smooth scrolling and element transitions using **GSAP**.
- **Modern UI/UX:** Crafted with **Tailwind CSS** for a clean, professional, and dark-themed aesthetic.
- **Dynamic Projects Showcase:** Highlighting my best work with detailed descriptions and links.
- **Fully Responsive:** Meticulously tested to ensure a "Pixel Perfect" experience across all devices (Mobile, Tablet, Desktop).
- **Contact Integration:** Easy way for recruiters and clients to get in touch.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) / [React.js](https://reactjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animation:** [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/)
- **Deployment:** [Vercel](https://vercel.com/)
- **Icons:** [Lucide React](https://lucide.dev/guide/)
